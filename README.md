# Учет коллекции марок (Stamp Collection Management)

## Описание проекта

Программный комплекс для учета и управления коллекцией марок, разработанный на Python с использованием объектно-ориентированного подхода. Приложение предоставляет удобный графический интерфейс для систематизации коллекции по альбомам, коллекциям и страницам.

## Основные возможности

- Иерархическое хранение данных: Альбомы - Коллекции - Страницы - Марки
- Полный CRUD-функционал: Добавление, редактирование, удаление
- Экспорт данных: В форматы XML и PDF
- Импорт данных: Из XML-файлов
- Валидация данных: Проверка корректности вводимой информации
- Логирование: Детальное журналирование всех операций

## Требования к системе
- Python 3.8 или выше
- MySQL Server 8.0 или выше
- Установленные библиотеки 

- Процессор: 1 ГГц или выше
- Оперативная память: 2 ГБ минимум
- Свободное место на диске: 100 МБ
